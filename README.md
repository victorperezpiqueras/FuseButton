# FuseButton
Interacción con objetos en UE4 para Google VR mediante un Fuse Button, desarrollado para la asignatura de Realidad Virtual y Aumentada.

 El fuse button permite interactuar con los objetos de la escena pasados 2 segundos. También se puede interactuar instantáneamente tocando la pantalla.
